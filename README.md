Uniwank
===============================================================================

From the Alhadis English Dictionary:

<dl><dt>Uniwank</dt
><dd><b>noun</b> <i>(uncountable, computing, vulgar)</i>
<ol><li>use of letter variants to create kitschy-looking
effects, especially for display in plain-text media with limited
or no text formatting. See also:
<a href="https://en.wiktionary.org/wiki/confusable#Noun">confusable</a>,
<a href="https://en.wiktionary.org/wiki/wankery#Noun">wankery</a>.
</li></ol></dd></dl>

You already know what this is. It's that distracting, annoying and pointless use
of Unicode mathematical symbols to create 𝐟𝐚𝐮𝐱-𝐛𝐨𝐥𝐝 or 𝒊𝒕𝒂𝒍𝒊𝒄 effects, often because
the medium within which the text appears doesn't support such formatting styles.

I wrote this tool after I unknowingly copied text to my code-editor that used an
effect like this for 𝚖𝚘𝚗𝚘𝚜𝚙𝚊𝚌𝚎𝚍 𝚌𝚑𝚊𝚛𝚊𝚌𝚝𝚎𝚛𝚜 (which, depending on your editor-font
of choice, might be completely indistinguishable in this very paragraph).

However, as a veteran shitposter, I would be remiss if I didn't also enable this
tool to apply the effects to ordinary text for use in YouTube or BlueSky comment
malarkey. Ergo, <b><samp>uniwank</samp></b> can apply every text transliteration
it's nominally designed to remove.
